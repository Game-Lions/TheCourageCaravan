# TheCourageCaravan

תנהיג שיירה של רכבי אספקה בדרך לירושלים הנצורה בתקופת מלחמת העצמאות. תקן רכבים שנפגעו, תלחם בפורעים שמנסים לעצור את השיירה ולספק עזרה לפצועים. האם תצליחו להגיע לירושלים בשלום עם כמה שיותר אספקה חיונית.
